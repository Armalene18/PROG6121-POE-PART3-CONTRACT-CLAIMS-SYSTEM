# Contact Claim System (CCS)

## Project Overview
The Contact Claim System (CCS) is a C# WPF application designed to streamline claim submissions, approvals, and HR processing. It provides a role-based workflow for four user types: Lecturer, Coordinator, Manager, and HR. The system automates calculations, tracks claim statuses, and allows users to upload supporting documents.  

This project also includes Word and PowerPoint documentation showcasing project requirements, lecturer feedback, and Part 3 improvements.


## Features
- **User Authentication & Role-Based Access:** Secure login with permissions based on user role.  
- **Claim Submission & Management:** Submit claims with attachments; view statuses (Pending, Approved, Rejected).  
- **Dashboard:** Clear summary of claims with visual indicators for status.  
- **Automated Calculations:** Claim payment amounts calculated automatically based on rules.  
- **Search & Filter:** Locate claims quickly by status or user.  
- **Data Persistence:** Claims and user data stored in JSON files.  
- **Error Handling & Validation:** Ensures input data is valid and prevents errors.  
- **Documentation & Presentation:** Includes Word report with feedback and Harvard references, and PowerPoint slides showcasing system workflow and improvements.
